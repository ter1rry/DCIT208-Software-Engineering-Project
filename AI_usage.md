# AI Usage Policy

## Purpose

This document outlines how Artificial Intelligence (AI) tools may be used during the development of this project.

## Approved AI Tools

Examples of approved AI tools include:

* ChatGPT
* GitHub Copilot
* Gemini
* Claude

## Acceptable Uses

AI tools may be used for:

* Brainstorming ideas
* Improving documentation
* Explaining concepts
* Assisting with code reviews
* Generating draft content

## Restrictions

* AI-generated content must be reviewed and verified by team members before use.
* Team members remain responsible for all submitted work.
* AI tools must not be used to misrepresent understanding or violate academic integrity policies.

## Documentation

Significant use of AI tools should be documented where required by the course guidelines.

## Agreement

All team members agree to use AI tools responsibly and in accordance with the University's academic integrity policies.
